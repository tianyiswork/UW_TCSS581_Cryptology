This is an implementation of RC4 in Python3 for TCSS581 Cryptology

Instructions:

1. Use terminal to go to the location of the script, RC4.py.
2. Feel free change the encryption key on line 88 in the script.
3. You can also to change the location of King James Bible on line 93 in the script.
4. Run 'python3 RC4.py' (if you are using Python3).
5. The encrypted cyphertext file will be outputted to the same directory as the script.
6. Use 'time python3 RC4.py', the runtime of this encryption is about 10 seconds.
